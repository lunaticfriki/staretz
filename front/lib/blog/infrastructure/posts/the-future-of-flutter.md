Flutter continues to mature with each release. WebAssembly (Wasm) support is bringing near-native performance to Flutter web by compiling Dart directly to Wasm rather than JavaScript.

Dart's type system keeps improving. Records and patterns added in Dart 3 enable more expressive code. Sealed classes make exhaustive pattern matching possible, which pairs perfectly with state machines.

The ecosystem of packages grows every month. The challenge shifts from finding packages to evaluating them — look at pub points, maintenance activity, and issue responsiveness before committing to a dependency.
